# PropertyAI - AI-Powered Real Estate Landing Page

A modern, fully animated landing page for PropertyAI, an AI firm that provides intelligent property insights for buyers and sellers. Built with the latest React and Next.js technologies with stunning dark theme animations.

## 🚀 Features

### Core Features
- **AI-Powered Market Intelligence** - Real-time property analysis and market insights
- **Smart Property Matching** - Find properties matching your criteria perfectly
- **Advanced Analytics Dashboard** - Comprehensive market visualizations
- **Price Prediction** - ML models predict property values with high accuracy
- **Neighborhood Insights** - Discover schools, amenities, and lifestyle factors
- **Expert AI Guidance** - Personalized recommendations for your needs

### Design & Animation Features
- **Dark Theme UI** - Modern dark aesthetic with cyan/blue gradient accents
- **Advanced Framer Motion Animations**
  - Smooth staggered entrance animations
  - Floating elements with parallax effects
  - Gradient background animations
  - Hover interactions with scale and glow effects
  - Rotating animated elements
  - Scroll-triggered animations
- **Fully Responsive** - Mobile-first design for all devices
- **Optimized Performance** - Built with Next.js 16 and Turbopack

## 🛠️ Tech Stack

- **React 18+** - Latest React features
- **Next.js 16** - Modern React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Advanced animation library
- **ESLint** - Code quality assurance

## 📁 Project Structure

```
src/
├── app/
│   ├── page.tsx          # Main landing page
│   ├── layout.tsx        # Root layout with metadata
│   └── globals.css       # Global styles with animations
├── components/
│   ├── Hero.tsx          # Hero section with animated gradient background
│   ├── Features.tsx      # Features grid with hover effects
│   ├── Testimonials.tsx  # Testimonials carousel
│   ├── Pricing.tsx       # Pricing plans with animations
│   ├── CTA.tsx           # Call-to-action section
│   └── Footer.tsx        # Footer with social links
```

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the landing page.

### Build

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
