## 🎨 Customization Quick Tips

### Change Heading Text
Edit `src/components/Hero.tsx`, find the `<h1>` tag

### Change Colors
Update Tailwind classes like `text-cyan-400` in components

### Add Your Logo
Place image in `public/` folder, import in components

### Update Pricing
Edit the `plans` array in `src/components/Pricing.tsx`

### Change Company Name
Update in `src/app/layout.tsx` metadata

### Add More Features
Edit the `features` array in `src/components/Features.tsx`

## 🐛 Troubleshooting

### Page not loading?
```bash
# Clear cache and rebuild
rm -rf .next node_modules
npm install
npm run dev
```

### Animations not working?
- Make sure you see `"use client"` at top of component files
- Check browser console for errors
- Try clearing browser cache

### Build errors?
```bash
npm run lint  # Check for TypeScript errors
npm run build # Full build check
```
